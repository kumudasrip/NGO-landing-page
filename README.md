# 🌟 Hope Foundation - NGO Landing Page

A modern, responsive landing page designed for non-profit organizations focused on children's education, food support, and community development.

## 🚀 Live Demo

**View Live Site:** https://kumudasrip.github.io/NGO-landing-page/

---

## ✨ Features

- **Fully Responsive Design** - Works seamlessly on mobile, tablet, and desktop
- **Modern UI/UX** - Clean, professional design with smooth animations
- **Bootstrap 5** - Built with the latest Bootstrap framework
- **Animated Statistics** - Counter animations triggered on scroll
- **Smooth Scrolling** - Enhanced navigation experience
- **SEO Friendly** - Optimized meta tags and semantic HTML
- **Fast Loading** - Optimized images and minimal dependencies
- **Easy Customization** - Well-commented code for easy modifications
- **Google Fonts Integration** - Beautiful typography with Playfair Display & Poppins
- **Bootstrap Icons** - Comprehensive icon library included

---

## 🛠️ Technologies Used

- **HTML5** - Semantic markup
- **CSS3** - Modern styling with flexbox and grid
- **Bootstrap 5.3.0** - Responsive framework
- **Bootstrap Icons** - Icon library
- **Google Fonts** - Custom typography
- **JavaScript** - Interactive features and animations
- **Unsplash** - High-quality placeholder images

## 📦 Installation

### Clone the Repository

```bash
# Clone this repository
git clone https://github.com/kumudasrip/NGO-landing-page.git

# Navigate to the project directory
cd ngo-landing-page

# Open in browser
open index.html
# or simply double-click the HTML file
```
